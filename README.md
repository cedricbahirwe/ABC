# ABC
## A Small Prototype using MultiPeerConnectivity in Swift and UIKIT, tested on iOS 12 and 13.

### Note that on ios 14.0 and above this app might require few changes to work due to apple new security policies. 

### Checkout [apple developer doc](https://developer.apple.com/documentation/multipeerconnectivity) for more details
#### A great and helpful resource to help you update this project can be found at [raywenderlich.com](https://www.raywenderlich.com/12689804-getting-started-with-multipeer-connectivity)
